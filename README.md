- Pandas & Numpy

Selengkapnya dapat dilihat pada [Requirements](requirements.txt)

### Cara Instalasi dari Github:
1. Memastikan Python sudah terpasang
2. Clone Repository ini
3. Masuk ke folder projek
4. Buat virtual environment (Windows)
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
5. Install Library yang dibutuhkan

   ```
   $ pip install -r requirements.txt
   ```

6. Menjalankan aplikasi

   ```
   $ streamlit run app.py
   ```
